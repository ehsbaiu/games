## 帮助
### 翻卡片游戏
1. 游戏刚开始加载时可能会出现乱码，稍微等待即可。
2. 点击卡片，寻找相同的卡片，全部点完时点击最上方的旋转符号或刷新页面即可重新分配卡片并重新开始。
